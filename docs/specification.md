# Hidden Pakistan — AI Powered Tourism Website

## Project Objective

The objective of this project is to develop an AI-powered tourism website for Pakistan using modern frontend and backend technologies with Retrieval-Augmented Generation (RAG) architecture.

## Technologies Used

- Next.js
- TypeScript
- Tailwind CSS
- FastAPI
- Python
- Scikit-learn
- TF-IDF Vectorization

## Features

- Responsive tourism website
- Dynamic destination cards
- AI tourism assistant
- RAG-based information retrieval
- Knowledge-base driven chatbot
- Frontend-backend integration

## AI Architecture

The project uses a Retrieval-Augmented Generation (RAG) pipeline:

User Query → TF-IDF Vectorization → Similarity Search → Knowledge Retrieval → Response Generation

## Deployment Plan

Frontend deployed using Vercel.
Backend executed using FastAPI server.
